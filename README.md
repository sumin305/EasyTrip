# EasyTrip ✈️
ChatGPT API를 활용한 똑똑한 여행 어플
개발기한 (2023.05 ~ 진행중)

##  프로젝트 소개 ⭐

### 주제
AI기반 여행 일정 추천 및 일기 어플 개발

### 주요 기능
- 1. ChatGPT API를 활용한 여행 계획 세우기
- 2. 사용자가 선택한 키워드로 여행 기록하기
